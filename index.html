▶ 🔍 How to use

• Use the category pills above to filter by tool type
• Click any boolean row label (✓/✗) to sort by that capability
• Use the search box to find tools by feature or description
• Toggle individual services and attributes on/off below
• Harness Strength (●●●●) reflects how autonomously the AI can act

▶ 📝 Notes

• "Fully autonomous" tools do real work while you watch (or don't)
• Coding agents (Claude Code, Codex) are powerful but require technical knowledge
• Harness quality matters more than model quality for most tasks — Gemini's model is just as smart, but its chatbot harness can't make Excel files
• Google Antigravity (coding agent) and Manus (standalone harness, now Meta) also exist but aren't yet widely used
• GPT-5.2 Pro is available at higher cost tiers — extremely capable for complex analytical work
• Suggestions welcome — fork it and make it better!
